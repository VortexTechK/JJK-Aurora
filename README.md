## Before You Complain
_**READ**_ the google doc before making pull requests; there are a lot of nuances. Don't waste both of our times. 

A lot of features are in equipment (ex. reverse curse technique) and many others are unimplemented for obvious reasons.

## Known Problems
 • Multiclass rules for innate techniques are weird and take your highest class which idk how to do. If you are planning to multiclass choose your innate technique on the class you plan to level the highest and choose that class first. <br>
 • Aurora only lets you display one companion so if your class+innate technique grants more than one (ex. shikigami user & auspicious beast summon), only the first one selected will appear. I recommend having a seperate place to track your companions/shikigami or just reference the stat block included in the class/innate technique<br>
